# Bioscope Plus 2.0 🎬

A clean, ad-free web application for movie reviews, ratings, and cinema information.

## Features ✨

- **No Advertisements** - Pure content experience
- **Movie Reviews** - Honest and detailed movie reviews
- **Movie Ratings** - Star ratings for movies
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Beautiful UI** - Modern gradient design with smooth animations
- **Contact Form** - Connect with us directly

## Technologies Used 🛠️

- HTML5
- CSS3 (Responsive, Gradients, Flexbox, Grid)
- Vanilla JavaScript (No frameworks)

## How to Use 🚀

1. Clone the repository
   ```bash
   git clone https://github.com/mrfajle01-commits/bioscope-plus-2.0.git
   ```

2. Open `index.html` in your browser

3. Or deploy it to GitHub Pages, Netlify, or any static hosting service

## File Structure 📁

```
bioscope-plus-2.0/
├── index.html      (Main HTML file)
├── styles.css      (Styling)
├── script.js       (JavaScript functionality)
└── README.md       (This file)
```

## Customization 🎨

- **Colors**: Edit the gradient colors in `styles.css`
- **Content**: Update text content in `index.html`
- **Movies**: Add more movie cards in the movies section
- **Reviews**: Add more reviews in the reviews section

## Deployment 🌐

### GitHub Pages
1. Go to repository Settings → Pages
2. Set source to `main` branch
3. Your site will be live at `https://mrfajle01-commits.github.io/bioscope-plus-2.0/`

### Other Hosting
- Netlify
- Vercel
- Any static hosting service
